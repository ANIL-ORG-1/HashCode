### Installation and use.

```sh
$ sudo pacman -S git // or sudo apt-get install git
$ git clone https://github.com/Sup3r-Us3r/HashCode.git
$ cd HashCode
$ sudo chmod +x hashcode.py
```
Use.
```sh
$ ./hashcode.py
```
